1. В корне проекта надо запустить hugo-сервер следующей командой: /hugo.exe server ;
- Эта команда запустит билд приложения на локальном сервере.
- Адрес, по которому можно будет посмотреть результат будет выведен в строке.

* В случае Windows для команд можно использовать cmd или Windows PowerShell (Hugo рекомендует).
